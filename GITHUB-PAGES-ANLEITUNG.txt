POKÉMONTYPLEARNER 0.6.1 AUF DEM IPHONE INSTALLIEREN

1. Erstelle auf github.com ein kostenloses Konto.
2. Erstelle ein neues Repository, zum Beispiel: PokemonTypLearner
3. Lade ALLE Dateien und Ordner aus diesem Projekt hoch – nicht nur index.html.
4. Öffne im Repository: Settings → Pages
5. Wähle unter „Build and deployment“: GitHub Actions
6. Warte nach dem Upload ungefähr 1–2 Minuten.
7. GitHub zeigt dir anschließend die Pages-Adresse an.

AUF DEM IPHONE:
1. Öffne die GitHub-Pages-Adresse in Safari.
2. Tippe auf das Teilen-Symbol.
3. Tippe auf „Zum Home-Bildschirm“.
4. Tippe oben rechts auf „Hinzufügen“.

WICHTIG:
- Die Installation funktioniert nicht direkt aus der ZIP-Datei.
- GitHub Pages liefert die App über HTTPS aus; dadurch funktionieren PWA und Offline-Cache.
- Bei einem späteren Update lädst du die neuen Dateien erneut ins Repository hoch.
