PokémonTypLearner – Version 0.6.1 Alpha

Dieser Patch macht die vorhandene Web-App vollständig für die Installation als PWA auf iPhone, Android und Desktop bereit.

ENTHALTEN
- iPhone-App-Icon in 180 × 180 Pixel
- getrenntes Maskable-Icon für Android
- verbessertes Web-App-Manifest
- stabilerer Offline- und Navigations-Fallback
- Installationsanleitung direkt in der App
- automatische Erkennung, wenn die App bereits installiert ist
- fertige GitHub-Pages-Deployment-Datei
- Anleitung GITHUB-PAGES-ANLEITUNG.txt
- Migration von Version 0.6 und älteren Spielständen

LOKALER TEST
Die index.html kann am PC geöffnet werden. Für echte Installation und Offline-Funktion muss das Projekt jedoch über HTTPS gehostet werden, z. B. mit GitHub Pages.
